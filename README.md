# ui-frontend-for-vr-face-recognition
 
ui-frontend-for-vr-face-recognition は、エッジコンピューティング環境下において、VR端末の視野内に写る人物を画像認識・分析して人物を特定し、VR端末の視覚内の見やすい領域に、テキストデータとして人物情報が表示される技術を含む、UIフロントエンド・マイクロサービスです。  　　

## 動作環境
・OS : Linux OS  
・CPU: ARM/AMD/Intel  
・Kubernetes  
・AION
